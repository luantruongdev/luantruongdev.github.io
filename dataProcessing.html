Summary

The Internet of Things relies on the deployment of sensor networks. These collect an immense amount of data to process and to extract the desired information. The so called study field Big Data is more and more popular today as it allows the automation of the analysis of large databases, in order to add value to the collected data.
The goal of this training unit was to provide the basics of analyzing large volumes of data and it consists of two modules:

  • Semantic web and data processing
  • Data processing and analysis: Big Data

Analysis, Data Processing and Business Applications

Processing Semantic Data

The semantic web were completely new to me, the lectures of this module allowed me to fully understand the usefulness and the possibilities offered by this field. The labs then allowed me to get started with a software named "Protégé". Thanks to this lab, I was able to create an ontology that I then enriched it with metadata in order to build the semantics and finally apply reasoner "Hermit" of Protégé to infer information of the ontology.

The second lab also allowed me to see how to enrich an ontology from Java code, which automates the task and allows me to work on a big amount of data. All the detailed description for these labs plus the source code on github are in the link below.

Data Processing and Analysis: Big Data

The practical work with R languge really allowed me to learn R and use it in concrete cases. This learning method suits me very well and i made good progress in mastering this language, after multiples sessions. Unlike the tutorials which were done alone, the project was done in pairs, which was an opportunity to share our methods and to think collectively on the best way to achieve the expected results. In this project we had to find datasets to analyze and then extract the relevant information.

As my teammate Michael and myself are NBA fans, we dicided to use Data Analysis to settle the biggest debate of our generation : Who between ​Lebron James​, ​Kobe Bryant​ and ​Michael Jordan​ can be considered the best player of our generation?


The dataset was retrieved from Kaggle regarding Jordan’s stats from 1984 to 2003, Bryant’s stats from 1996 to 2006 and James’ stats from 2003 to 2019. For each game we have information about the minutes played, the points scored, assists, rebounds etc.. Link of our detailed report can be found below.
